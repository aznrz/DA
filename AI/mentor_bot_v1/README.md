# 🤖 AI Mentor Bot (Python & Excel)

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![Ollama](https://img.shields.io/badge/LLM-Ollama-orange.svg)](https://ollama.ai/)
[![Telegram](https://img.shields.io/badge/Bot-Telegram-blue.svg)](https://core.telegram.org/bots)

Персональный ИИ-ассистент для аналитиков данных. Бот развернут локально и помогает быстро решать задачи по **Python (Pandas, Numpy)**, **SQL** и **Excel**, не отправляя ваши данные в облако.

---

## 🚀 Основные возможности

* **Полная приватность**: Благодаря использованию [Ollama](https://ollama.ai/), все вычисления проходят на вашем локальном железе.
* **Контекстная память**: Бот хранит историю текущего диалога, что позволяет задавать уточняющие вопросы и работать над кодом итеративно.
* **Лаконичный режим**: Системный промпт настроен на выдачу максимально конкретных ответов — только суть, готовый код или формулы Excel.
* **Управление памятью**: Команда `/clear` мгновенно очищает контекст для начала новой задачи.

## 🧠 Технический стек и выбор модели

Для проекта выбрана модель **Mistral 7B**, как наиболее сбалансированное решение для задач технического консультирования.

### Почему Mistral 7B?
Модель с 7 миллиардами параметров обеспечивает высокую точность в написании кода при умеренных требованиях к ресурсам. На системе с **16 ГБ RAM** модель работает плавно, позволяя параллельно использовать тяжелые инструменты (Power BI, Excel, IDE).

### Квантование (Quantization)
В проекте используются квантованные версии моделей. Это позволяет запускать мощные нейросети на обычном пользовательском железе:

| Уровень (Bit) | Требуемая RAM | Рекомендация |
| :--- | :--- | :--- |
| **Q3 (3-bit)** | < 8 ГБ | Для слабых систем (возможны ошибки в синтаксисе). |
| **Q4 (4-bit)** | **8–10 ГБ** | **Оптимально (выбор по умолчанию).** Баланс скорости и точности. |
| **Q8 (8-bit)** | 12–16 ГБ | Максимальная точность для сложных архитектурных задач. |
