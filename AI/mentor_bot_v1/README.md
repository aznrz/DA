* **Локальный и приватный запуск**: Благодаря [Ollama](https://ollama.ai/), все вычисления проходят на вашем локальном железе, данные не покидают ваш компьютер.

* **Режим лаконичности**: Системный промпт настроен на выдачу максимально конкретных ответов — только суть, готовый код или формулы Excel.

## 🧠 Выбор модели: Mistral 7B

Для проекта выбрана модель **Mistral 7B**, как наиболее сбалансированное решение по скорости и точности для технических вопросов.

### Техническое обоснование
* **RAM**: Для комфортной работы требуется 8–10 ГБ свободной оперативной памяти. На системе с **16 ГБ RAM** модель работает плавно, позволяя параллельно использовать IDE и браузер.
* **Квантование (Quantization)**: По умолчанию используется версия **Q4 (4-bit)**. Это снижает вес модели до ~4.1 ГБ без значительной потери качества ответов.

### Рекомендации по выбору версии (Quantization)
| Уровень | Требуемая RAM | Описание |
| :--- | :--- | :--- |
| **Q3 (3-bit)** | < 8 ГБ | Для слабых систем (возможны ошибки в синтаксисе). |
| **Q4 (4-bit)** | **8–10 ГБ** | **Оптимально.** Баланс скорости и точности. |
| **Q8 (8-bit)** | 12–16 ГБ | Максимальная точность, требует больше ресурсов CPU. |

## 🛠️ Как запустить

### 1. Установите Ollama
Скачайте и установите с [ollama.com](https://ollama.com/). Затем загрузите модель командой:
```bash
ollama pull mistral
```

### 2. Подготовьте окружение
Клонируйте репозиторий и установите зависимости:
```bash
pip install -r requirements.txt
```

### 3. Настройте переменные окружения
Создайте файл `.env` на основе `.env.example`:
```env
TELEGRAM_TOKEN=ваш_токен_от_BotFather
OLLAMA_URL=http://localhost:11434/api/chat
MODEL_NAME=mistral
```
> **Важно:** Файл `.env` добавлен в `.gitignore` и не должен публиковаться в открытом доступе.

### 4. Запустите бота
```bash
python bot.py
```
