# AI Mentor Bot (Python & Excel)

Telegram-бот на базе LLM (Ollama), помогающий аналитикам данных быстро получать ответы по Python (Pandas) и Excel.

## Особенности
- **Локальный запуск**: Использует Ollama и модель Mistral (бесплатно и приватно).
- **Контекст**: Бот помнит последние несколько сообщений в диалоге.
- **Лаконичность**: Настроен давать короткие ответы без лишней «воды».

## Как запустить
1. Установите [Ollama](https://ollama.ai/) и скачайте модель: `ollama pull mistral`.
2. Клонируйте репозиторий.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Создайте файл `.env` на основе `.env.example` и вставьте ваш токен от BotFather.
5. Запустите: `python bot.py`.
