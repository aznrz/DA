## AI Mentor Bot (Python & Excel)
Telegram-бот на базе локальной LLM (Ollama), предназначенный для оперативной помощи аналитикам данных в вопросах Python (Pandas, Numpy), SQL и Excel.

## 🚀 Особенности
Локальный и приватный запуск: Бот работает через Ollama, поэтому ваши данные не покидают ваш компьютер.

Контекстная память: Бот помнит историю текущего диалога, что позволяет уточнять запросы и поэтапно разбирать код.

Режим лаконичности: Настроен давать максимально короткие ответы, фокусируясь на готовых формулах и блоках кода.

## 🧠 Выбор модели: Mistral 7B
В данном проекте используется модель Mistral 7B, так как она обеспечивает оптимальный баланс между скоростью генерации и точностью ответов на технические вопросы.

## Техническое обоснование
RAM: Для комфортной работы версии 7B требуется 8–10 ГБ свободной оперативной памяти. На системе с 16 ГБ RAM модель работает плавно, оставляя ресурсы для IDE и браузера.

## Квантование (Quantization): По умолчанию Ollama использует версию Q4 (4-bit), что снижает вес модели до ~4.1 ГБ без значительной потери качества.

## Рекомендации по выбору версии (Quantization)
Если вы хотите адаптировать бота под другие ресурсы, обратите внимание на типы сжатия:

Q3 (3-bit): Для систем с памятью менее 8 ГБ (может ошибаться в синтаксисе).

Q4 (4-bit): Рекомендуемый стандарт для 16 ГБ RAM (оптимальная скорость и точность).

Q8 (8-bit): Максимальная точность, требует более 12 ГБ RAM и может работать медленнее на CPU.

🛠️ Как запустить
Установите Ollama:
Скачайте и установите с ollama.com. Затем загрузите модель командой:

Bash
ollama pull mistral
Подготовьте окружение:
Клонируйте репозиторий и установите зависимости:

Bash
pip install -r requirements.txt
Настройте переменные окружения:
Создайте файл .env на основе .env.example:

Фрагмент кода
TELEGRAM_TOKEN=ваш_токен_от_BotFather
OLLAMA_URL=http://localhost:11434/api/chat
MODEL_NAME=mistral
Важно: файл .env добавлен в .gitignore и не должен публиковаться в открытом доступе.

Запустите бота:

Bash
python bot.py
