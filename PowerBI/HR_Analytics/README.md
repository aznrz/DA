# 📊 HR Analytics Dashboard (Power BI)

## 📌 Описание проекта

Интерактивный HR-дашборд для анализа текучести персонала, демографии сотрудников и ключевых факторов увольнений.

Проект позволяет:
- отслеживать текучесть кадров  
- анализировать сотрудников по возрасту, стажу и зарплате  
- выявлять проблемные зоны в компании  
- принимать data-driven HR решения  

---

## 🎯 Цель проекта

- Оценить уровень текучести персонала  
- Найти причины увольнений  
- Определить группы риска  
- Повысить прозрачность HR-метрик  

---

## 🛠️ Стек технологий

- **Power BI** (DAX, Data Modeling)  
- **Power Query (M)** — трансформация данных  
- **SQL (PostgreSQL)** — хранение данных  
- **Excel / CSV** — источник данных  
- **SharePoint** — хранение и доступ к данным  
- **Power Automate** — автоматизация обновлений

## 📊 Основные метрики

- Количество сотрудников  
- Количество уволившихся  
- Текучесть персонала (%)  
- Средний возраст  
- Средняя зарплата

## 🧮 DAX расчёты

### Количество сотрудников

```DAX
Кол-во сотрудников =
DISTINCTCOUNT(HR_Analytics_excel[EmpID])

### Уволившиеся сотрудники

```DAX
Уволилось =
CALCULATE(
    [Кол-во сотрудников],
    NOT ISBLANK(HR_Analytics_excel[Дата увольнения])
)
```

### Текучесть персонала

```DAX
% Текучести персонала =
DIVIDE(
    [Уволилось],
    [Кол-во сотрудников],
    0
)
```

### Средний возраст

```DAX
Средний возраст =
AVERAGE(HR_Analytics_excel[Age])
```

### Средняя зарплата

```DAX
Средняя зарплата =
AVERAGE(HR_Analytics_excel[MonthlyIncome KZT])
```

### Средний стаж

```DAX
Стаж Срдн. =
AVERAGE(HR_Analytics_excel[YearsAtCompany])
```

## 🔍 Инсайты

* Основная текучесть наблюдается среди сотрудников с небольшим стажем
* Наиболее уязвимый период — первые годы работы
* Некоторые должности имеют значительно более высокий уровень текучести
* Сотрудники с более низкой зарплатой чаще увольняются
* Удовлетворенность сотрудников напрямую влияет на удержание
* Возрастная группа 26–35 наиболее мобильна
