## 📌 Описание проекта

Аналитический дашборд для исследования рынка вакансий аналитиков на основе данных HeadHunter API.

Проект позволяет анализировать:

* динамику спроса на аналитиков
* уровень заработных плат
* географию вакансий
* распределение по ролям и опыту
* **востребованные навыки (skills)**

---

## 🎯 Цель проекта

Определить ключевые тенденции рынка аналитиков:

* как меняется спрос со временем
* какие зарплаты предлагаются
* какие роли наиболее востребованы
* в каких городах сосредоточен спрос
* **какие навыки требуются работодателями чаще всего**

## 🧮 DAX расчёты

### 1. Salary Row (KZT)

```DAX
Salary Row (KZT) = 
VAR FromVal = 'final_df'[Зарплата От]
VAR ToVal = 'final_df'[Зарплата До]
VAR Rate = COALESCE('final_df'[Rate_Column], 1)

VAR RowSalary =
    IF(
        NOT ISBLANK(FromVal) && NOT ISBLANK(ToVal),
        (FromVal + ToVal) / 2,
        COALESCE(FromVal, ToVal)
    )

RETURN RowSalary * Rate
```

### 2. Средняя зарплата

```DAX
Зарплата средняя (KZT) = 
AVERAGE('final_df'[Salary Row (KZT)])
```

### 3. Медианная зарплата

```DAX
Зарплата Медиана (KZT) = 
MEDIAN('final_df'[Salary Row (KZT)])
```

## 🔍 Инсайты
Основной спрос сосредоточен в Алматы (~63%) и Астане (~25%)
Рынок ориентирован на middle-специалистов (1-6 лет опыта ≈ 89%)
BI-аналитики получают более высокие зарплаты
Наблюдается рост количества вакансий со временем
Значительная часть вакансий не содержит зарплату

## 💡 Инсайт по навыкам и зарплатам
Вакансии, требующие A/B testing, Airflow и ETL, демонстрируют более высокую медианную заработную плату по сравнению с другими навыками
Это указывает на то, что:
данные навыки относятся к более продвинутому уровню аналитики
работодатели готовы платить больше за специалистов с опытом работы с данными pipeline и экспериментами
Особенно выделяется:
- A/B testing — один из самых высокооплачиваемых навыков
- Airflow — востребован в data engineering / advanced analytics
- ETL — ключевой навык для работы с хранилищами данных

📌 Вывод:
Освоение инструментов работы с данными (ETL, Airflow) и методов анализа (A/B testing) значительно повышает рыночную стоимость специалиста.
