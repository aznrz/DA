# 📊 HeadHunter Analytics Dashboard (Power BI)

## 📌 Описание проекта
Аналитический дашборд для исследования рынка вакансий аналитиков на основе данных HeadHunter API.

Проект позволяет анализировать:
- динамику спроса на аналитиков
- уровень заработных плат
- географию вакансий
- распределение по ролям и опыту

---

## 🎯 Цель проекта
Определить ключевые тенденции рынка аналитиков:

- как меняется спрос со временем
- какие зарплаты предлагаются
- какие роли наиболее востребованы
- в каких городах сосредоточен спрос

---

## 🛠️ Стек технологий

- **Power BI** (DAX, Data Modeling)
- **Python** (API, ETL pipeline)
- **Excel** (справочник стоп-слов)
- **REST API** (HeadHunter)

---

## 🐍 Data Pipeline

Реализован ETL-процесс:

- сбор данных через HeadHunter API
- фильтрация нерелевантных вакансий (stop-words)
- инкрементальная загрузка данных
- накопление исторических данных
- определение статуса вакансий (Активна / В архиве)

---

## 📊 Основные метрики

- Количество вакансий  
- Средняя зарплата (KZT)  
- Медианная зарплата  
- Количество компаний  
- Доля вакансий с указанной зарплатой  
- Динамика вакансий по датам

## ⚙️ Архитектура данных

Модель построена по принципу **звезды (Star Schema)**:

- **FactVacancies** — факты по вакансиям  
- **DimDate** — календарь  
- **DimCompany** — компании  
- **DimLocation** — города

## 🧮 DAX расчёты

### Средняя зарплата (с учётом валюты)

```DAX
Зарплата средняя (KZT) =
AVERAGEX(
    'final_df',
    VAR FromVal = 'final_df'[Зарплата От]
    VAR ToVal = 'final_df'[Зарплата До]
    VAR Rate = COALESCE('final_df'[Rate_Column], 1)

    VAR RowSalary =
        IF(
            NOT ISBLANK(FromVal) && NOT ISBLANK(ToVal),
            (FromVal + ToVal) / 2,
            COALESCE(FromVal, ToVal)
        )

    RETURN RowSalary * Rate
)
```
)

📌 Особенности:

учитывает валютный курс

корректно обрабатывает вилки зарплат

зависит от фильтров (город, роль, опыт)

🔍 Инсайты

Наибольший спрос на аналитиков наблюдается в крупных городах

Вакансии BI-аналитиков имеют более высокий уровень зарплат

Наблюдается рост количества вакансий во времени

Не все вакансии содержат информацию о зарплате
