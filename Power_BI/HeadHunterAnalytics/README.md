# 📊 HeadHunter Analytics Dashboard (Power BI)

## 📌 Описание проекта
Аналитический дашборд для анализа рынка вакансий аналитиков на основе данных HeadHunter API.

Проект охватывает:
- динамику вакансий
- анализ зарплат
- географию спроса
- фильтрацию по ролям и опыту

---

## 🎯 Цель
Понять:
- как меняется спрос на аналитиков
- какие зарплаты предлагаются
- какие роли наиболее востребованы

---

## 🛠️ Стек технологий

- Power BI (DAX, Data Modeling)
- Python (API parsing)
- Excel (справочник стоп-слов)
- REST API (HeadHunter)

---

## 🔌 Источник данных

Данные получены через API HeadHunter с использованием Python-скрипта.

Реализовано:
- параметр поиска (ключевые слова)
- очистка данных (stop words)
- накопление исторических данных

---

## 📊 Основные метрики

- Количество вакансий
- Средняя зарплата (KZT)
- Медианная зарплата
- Количество компаний
- Динамика по датам

---

## 📈 Визуализации

- 📉 Динамика вакансий по времени
- 📊 Распределение зарплат
- 🗺 География вакансий
- 📌 Фильтры по:
  - городу
  - опыту
  - вакансии
  - компании

---

## ⚙️ Архитектура данных

Модель построена по принципу звезды:

- FactVacancies
- DimDate
- DimCompany
- DimLocation
