## 📌 Описание проекта
Аналитический дашборд для исследования рынка вакансий аналитиков на основе данных HeadHunter API.

Проект позволяет анализировать:
- динамику спроса на аналитиков
- уровень заработных плат
- географию вакансий
- распределение по ролям и опыту

---

## 🎯 Цель проекта
Определить ключевые тенденции рынка аналитиков:

- как меняется спрос со временем
- какие зарплаты предлагаются
- какие роли наиболее востребованы
- в каких городах сосредоточен спрос

## 🧮 DAX расчёты

### 1. Salary Row (KZT) для расчета
```
Salary Row (KZT) = 
VAR FromVal = 'final_df'[Зарплата От]
VAR ToVal = 'final_df'[Зарплата До]
VAR Rate = COALESCE('final_df'[Rate_Column], 1)

VAR RowSalary =
    IF(
        NOT ISBLANK(FromVal) && NOT ISBLANK(ToVal),
        (FromVal + ToVal) / 2,
        COALESCE(FromVal, ToVal)
    )

RETURN RowSalary * Rate
```

### 2. Средняя зарплата (с учетом валюты)

```DAX
Зарплата средняя (KZT) = 
AVERAGE('final_df'[Salary Row (KZT)])
```

### 3. Зарплата до (KZT)
```DAX
Зарплата до (KZT) = 
AVERAGEX(
    FILTER(
        'final_df',
        NOT ISBLANK('final_df'[Зарплата До])
    ),
    'final_df'[Зарплата От] * COALESCE('final_df'[Rate_Column], 1)
)
```
### 4. Зарплата от (KZT)
```
Зарплата от (KZT) = 
AVERAGEX(
    FILTER(
        'final_df',
        NOT ISBLANK('final_df'[Зарплата От])
    ),
    'final_df'[Зарплата От] * COALESCE('final_df'[Rate_Column], 1)
)
```

### 5. Зарплата Медиана (KZT)
```
Зарплата Медиана (KZT) = 
MEDIAN('final_df'[Salary Row (KZT)])
```

📌 Особенности:

- учитывает валютный курс
- корректно обрабатывает вилки зарплат
- зависит от фильтров (город, роль, опыт)

🔍 Инсайты

## 🔍 Инсайты

- Наибольший спрос на аналитиков сосредоточен в крупных городах, прежде всего в Алматы и Астане  
- Алматы является ключевым центром рынка: около **63% вакансий**, тогда как на Астану приходится около **25%**  

- Вакансии BI-аналитиков в среднем предлагают более высокий уровень заработной платы по сравнению с другими ролями  

- Наблюдается рост количества вакансий во времени, что указывает на увеличение спроса на специалистов  

- Значительная часть вакансий не содержит информации о заработной плате, что снижает прозрачность рынка  
- Крупные компании (особенно в финтехе), такие как Bereke Bank, Freedom и Народный банк, чаще не указывают уровень зарплаты  

### 💰 Анализ зарплат по опыту

- Без опыта: медианная зарплата около **280 тыс. KZT**  
- 1–3 года: около **430 тыс. KZT** (**45% всех вакансий**)  
- 3–6 лет: около **600 тыс. KZT** (**44% вакансий**)  
- 6+ лет: около **700 тыс. KZT**  

📌 Основной спрос сосредоточен на специалистах с опытом **1–6 лет (~89% вакансий)**, что указывает на высокий спрос на специалистов middle-уровня
