# 📊 Manager Performance Dashboard (Power BI)

## 📌 Описание проекта

Интерактивный BI-дашборд для анализа эффективности менеджеров по продажам.  
Проект позволяет оценивать выполнение планов, анализировать динамику продаж и выявлять ключевые факторы роста бизнеса.

---

## 🎯 Цель проекта

- Оценить эффективность менеджеров (KPI)
- Сравнить фактические продажи с планом
- Выявить лидеров и зоны роста
- Проанализировать динамику продаж
- Определить вклад регионов и партнёров

---

## 🛠️ Стек технологий

- **Power BI** (DAX, Data Modeling)  
- **Power Query (M)** — подготовка и трансформация данных  
- **SQL (PostgreSQL)** — работа с базами данных  
- **Python** — обработка данных и автоматизация  
- **SharePoint** — хранение и управление данными  
- **Power Automate** — автоматизация ETL и обновления данных  
- **Excel / CSV** — источники данных  

---

## 🐍 Data Pipeline

Реализован полный цикл обработки данных:

- загрузка данных из SQL / Excel / SharePoint  
- очистка и трансформация данных (Power Query)  
- автоматизация обновлений через Power Automate  
- моделирование данных в Power BI  
- визуализация и анализ  

---

## 📊 Основные метрики

- **Sales** — фактические продажи  
- **Plan** — план продаж  
- **KPI (%)** — выполнение плана  
- **To Plan %** — отклонение от плана  
- **Fact - Plan** — абсолютное отклонение  

---

## 📈 Визуализации

### 👨‍💼 Анализ менеджеров
- KPI по каждому менеджеру  
- Ранжирование по эффективности  
- Выявление лидеров и аутсайдеров  

### 📅 Динамика продаж
- Сравнение **Sales vs Plan** по месяцам  
- Анализ сезонности  

### 🗺 Анализ регионов
- Распределение продаж по городам  
- Определение ключевых регионов  

### 🤝 Анализ партнёров
- Продажи по партнёрам  
- Сегментация по бизнес-направлениям  

---

## ⚙️ Архитектура данных

Модель построена по принципу **звезды (Star Schema)**:

- **FactSales** — факты продаж и планов  
- **DimDate** — календарь  
- **DimManager** — менеджеры  
- **DimLocation** — регионы  
- **DimPartner** — партнёры
